---

# 1. 数据表总览

| 表名 | 说明 |
|---|---|
| 活动配置表 | 活动全局配置 |
| 每日配置表 | 每日配置表（含赛事、社交任务、赠送世界币、抽奖） |
| 赛程表 | 赛程主表（第三方 API 同步） |
| 竞猜记录表 | 用户竞猜记录 |
| 社交任务记录表 | 用户社交任务完成记录 |
| 抽奖记录表 | 用户抽奖记录 |
| 世界币流水表 | 世界币流水（含期末余额） |
| 抽奖奖品配置表 | 抽奖奖品概率配置 |
| 合约任务配置表 | 合约交易梯度任务配置 |
| 合约任务领取记录表 | 用户每日合约任务领取记录 |
| 签到记录表 | 用户每日签到记录及连签天数 |
| 用户活动统计表 | 用户活动期合约交易额汇总（排行榜积分） |
| 用户每日快照表 | 每日 UTC 12:00 快照，供落地页展示世界币 / 排名 / 合约交易额变化量 |
| 奖励记录表 | 奖励记录（含审核状态） |

# 4. 赛程表

第三方 API 同步的赛程信息，运营配置竞猜时从此表选赛程。

| 字段 | 类型 | 说明 |
|---|---|---|
| 赛事ID | string | 赛事唯一 ID |
| 主队名称 | string |  |
| 客队名称 | string |  |
| 开赛时间 | timestamp |  |
| 赛段 | string | 来源 API stage 字段，英文枚举，如 GROUP_STAGE / LAST_16 / QUARTER_FINALS / SEMI_FINALS / THIRD_PLACE / FINAL |
| 分组 | string | 来源 API group 字段，英文枚举，如 GROUP_A … GROUP_L；淘汰赛阶段为空 |
| 状态 | enum | 待开赛 / 进行中 / 已结束 |
| 赛果 | enum | 主胜 / 平局 / 客胜 |
| 最后同步时间 | timestamp |  |
